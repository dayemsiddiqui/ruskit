Welcome to the Rustavel documentation. Here you'll find comprehensive guides and documentation to help you start working with Rustavel as quickly as possible.

What is Rustavel?

Rustavel is a web application framework for Rust with expressive, elegant syntax. We believe development must be an enjoyable and creative experience to be truly fulfilling. Rustavel takes the pain out of development by easing common tasks used in many web projects.

Why Rustavel?

  • Built on top of battle-tested Rust web frameworks
  • Elegant syntax inspired by Laravel
  • Robust database integration with SeaORM
  • Powerful CLI tools for rapid development
  • Modern templating with Askama
  • Built-in authentication and authorization
